Student Managements System

I created a student Registration system that makes it easy for users (administrators) to enter and manage student information. This system collects important details like name, class, student ID, mobile number, and email address. Once entered, this information is stored securely and can be easily viewed whenever needed. Users can add new students or delete existing ones, making it simple to handle student registrations efficiently. The project is designed to improve how student data is organized and accessed, making administrative tasks much easier.
